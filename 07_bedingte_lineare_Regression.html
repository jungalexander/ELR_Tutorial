Bedingte lineare Regression

Input

Wird EffectLiteR geffnet, ein Datensatz eingelesen und unter Manifest Variables im Eingabe-Panel eine abhngige Variable, eine kategoriale Treatmentvarible sowie eine oder mehrere kategoriale oder kontinuierliche Kovariate(n) definiert (ausfhrliche Beschreibung zum Input siehe Input der einfachen linearen Regression), so schtzt EffectLiteR ein bedingtes lineares Regressionsmodell. Die bedingte lineare Regression ist die Defaulteinstellung von EffectLiteR.
Sollten zuvor unter dem Reiter Interactions im Eingabe-Panel nderungen vorgenommen worden sein, um beispielsweise eine zweifache lineare Regression zu schtzen, so kann die bedingte lineare Regression durch ein Klick auf Full model wieder hergestellt werden.

Parametrisierung

Anders als die zweifache lineare Regression lsst die bedingte lineare Regression eine Interaktion zwischen Treatmenteffekt und Kovariaten zu. Dies bedeutet, dass fr jede geschtzte Interaktion (abhngig von der Anzahl der Stufen der Treatmentvariablen X, der Anzahl an Kovariaten, und der Anzahl an Stufen von kategorialen Kovariaten) ein weiterer Parameter in das Regressionsmodell aufgenommen wird. Auch fr die bedingte lineare Regression betrachten wir einfache Beispiele mit einer kategorialen bzw. einer kontinuierlichen Kovariaten.

Zweistufige Kovariate, zweistufige Treatmentvariable

Wir verwenden zur Veranschaulichung das gleiche Beispiel, dass wir schon bei der zweifachen linearen Regression verwendet haben, nur lassen wir jetzt die Annahme auen vor, dass es keine Interaktion zwischen Treatment-Variable und Kovariate gibt:

Beispiel 5
Abhngige Variable \(\small Y\): Gesundheitsscore von Patienten.
Treatmentvariable \(\small X\): Erhalten Patienten eine (\(\small X=1\)) oder erhalten sie keine Behandlung (\(\small X=0\)).
Kovariate \(\small K\): Patient gibt an in seinem Umfeld ausreichend soziale Untersttzung zu erhalten (\(\small K=1\)) bzw. nicht genug Untersttzung zu erhalten (\(\small K=0\)).

\[\begin{align} g_{000} = E(Y|X=0, K=0) (\#eq:bed1) \end{align}\]

Der Parameter g000 schtzt die Ausprgung des Regressanden unter der Bedingung \(\small X=0\) und \(\small K=0\), siehe Gleichung @ref(eq:zwei2).

\[\begin{align} g_{010} = E(Y|X=0,K=1) - E(Y|X=0,K=0) (\#eq:bed2) \end{align}\]

Der Parameter g100 schtzt den Treatmenteffekt, den eine Vernderung von \(\small X=0\) zu \(\small X=1\) unter der Bedingung \(\small K=0\) auf \(\small Y\) ausbt, siehe Gleichung @ref(eq:zwei4). Beachte auch hier, dass die Bedingung \(\small K=0\), anders als bei der zweifachen linearen Regression, von Bedeutung ist.

Um den Parameter g110 inhaltlich zu verstehen, leiten wir in einem ersten Schritt her, wie der Parameter durch Erwartungswerte ausgedrckt werden kann. Parameter g110 mit Erwartungswerten ausdrcken

Der Parameter g110 ist Teil der Effektfunktion \(\small g1(k)\), auerdem wird er mit der Indikatorvariable \(I_{K=1}\) multipliziert (siehe Gleichung @ref(eq:bed4): Modellgleichung der bedingten linearen Regression mit zweistufiger Kovariaten und zweistufiger Treatmentvariablen).

\[\begin{align} E(Y|X,K) &= g0(K)+g1(K)\times I_{X=1} \\ &=g_{000} + g_{010} \times I_{K=1} + (g_{100} + g_{110} \times I_{K=1}) \times I_{X=1} (\#eq:bed4) \end{align}\]

Unter der Bedingung \(X=0\) wird die Effektfunktion mit Null multipliziert, under der Bedingung \(K=0\) wird der Parameter g110 mit Null multipliziert, das bedeutet die einzige Bedingung, unter der der Parameter g110 nicht mit Null multipliziert wird, ist die Bedingung \(\small X=1\) und \(\small K=1\).

Betrachten wir die Bedingung \(\small X=1\) und \(\small K=1\) und setzen fr alle drei bisher hergeleiteten Parameter g000, g010 und g100 ihre Erwartungswerte (bzw. Differenzen aus Erwartungswerten) ein, so knnen wir auch den Parameter g110 mit Erwartungswerten ausdrcken (Gleichungen @ref(eq:bed5) und @ref(eq:bed6):

\[\begin{align} E(Y|X=1,K=1)=& g_{000} + g_{010} \times I_{K=1} + (g_{100} + g_{110} \times I_{K=1}) \times I_{X=1}\notag\\ =& g_{000} + g_{010} \times 1 + (g_{100} + g_{110} \times 1) \times 1\notag\\ =& E(Y|X=0,K=0) + [E(Y|X=0,K=1) - E(Y|X=0,K=0)]+ \notag\\ & + [E(Y|X=1,K=0) - E(Y|X=0,K=0)] + g_{110} (\#eq:bed5) \end{align}\]

Zwei der Erwartungswerte in Gleichung @ref(eq:bed5), dritten Zeile, knnen herausgekrzt werden, anschlieend g110 freigestellt werden:

\[\begin{align} E(Y|X=1,K=1) =& E(Y|X=0,K=1) + [E(Y|X=1,K=0) - E(Y|X=0,K=0)] +\\ & +g_{110}\\ g_{110} =& [E(Y|X=1,K=1) - E(Y|X=0,K=1)] -\\ &-[E(Y|X=1,K=0) - E(Y|X=0,K=0)]\\ (\#eq:bed6) \end{align}\]

Inhaltlich bedeutet dies: Der Parameter g110 schtzt wie sehr der Effekt von \(\small X\) auf \(\small Y\) unter der Bedingung \(\small K=1\) \(\small [E(Y|X=0,K=1) - E(Y|X=1,K=1)]\) vom Effekt von \(\small X\) auf \(\small Y\) unter der Bedingung \(\small K=0\) \(\small [E(Y|X=1,K=0) - E(Y|X=0,K=0)]\) abweicht. Bzw: Der geschtzte Effekt von \(\small X\) auf \(\small Y\) ist unter der Bedingung \(\small K=1\) um den Parameter g110 grer als unter der Bedingung \(\small K=0\).

Beachte: Wenn du eine bedingte lineare Regression und eine zweifache lineare Regression fr den jeweils selben Datensatz durchfhrt und es keine Interaktion gibt (wenn die deskriptive Interaktion unabhngig von ihrer Signifikanz exakt den Wert Null annimmt), so sind die Parameter g000, g010 und g100, die in beiden Fllen geschtzt werden, identisch. Wenn im Datensatz allerdings Interaktionen auftreten werden sich die geschtzten Parameter in beiden Fllen unterscheiden.

Parameter g110 als Interaktion verstehen

Eine Interaktion drckt aus, dass ein Effekt unter bestimmten Bedingungen unterschiedlich ausfllt. Bezogen auf unser Beispiel heit das, dass der Effekt einer Behandlung auf den Gesundheitsscore von Patienten unter der Bedingung subjektiv zu geringer sozialer Untersttzung \(\small (K=0)\) evtl. anders ausfllt als unter der Bedingung subjektiv ausreichender sozialer Untersttzung \(\small (K=1)\). Um zu prfen ob sich der Treatmenteffekt zwischen den beiden Bedingungen unterscheidet, bilden wir die Differenz beider Effekte: Ist die Differenz nicht gleich Null, so unterscheiden sich die Effekte unter den verschiedenen Bedingungen zumindest deskriptiv voneinander. Den Treatment-Effekt unter der Kovariaten-Bedingung \(\small K=0\), beschrieben durch den Parameter g100, hatten wir bisher wie folgt ausgedrckt: \(\small E(Y|X=1,K=0) - E(Y|X=0,K=0)\).
Den Treatmenteffekt unter der Bedingung \(\small K= 1\) (nicht durch einen eigenen Parameter abgebildet) knnte man synonym dazu wie folgt ausdrcken: \(\small E(Y|X=1,K=1) - E(Y|X=0,K=1)\).
Subtrahiert man den Treatmenteffekt unter der Bedingung \(\small K=0\) vom Treatmenteffekt unter der Bedingung \(\small K= 1\), so testet man, ob die Treatmenteffekte unter beiden Bedingungen identisch sind, bzw.: Man beschreibt die Interaktion aus Treatment (\(\small X\)) und Kovariate (\(\small K\)). Auf diesem Wege erhlt man den selben Term, den wir auch schon in Gleichung @ref(eq:bed6) zuvor hergeleitet haben:

\[\begin{align} IA_{X \times K} =& E(Y|X=1,K=1) - E(Y|X=0,K=1) [E(Y|X=1,K=0) - E(Y|X=0,K=0)]\\ =& g_{110} (\#eq:bed7) \end{align}\]

wurde die Kovariate \(\small Z_1\) zuvor z-Standardisiert ist der Parameter g101 wie folgt zu interpretieren: Eine Vernderung der Kovariaten \(\small Z_1\) um plus eine Standardabweichung verndert den Effekt des Treatments auf die abhngige Variable um den Parameter g101. Wurde die Kovariate nicht z-Standardisiert, so drckt der Parameter die erwartete Steigerung des Erwartungswert bei einer Steigerung der Kovariaten um eine Einheit ihrer Skala aus. Hypothesentest 1

Getestet wird die Nullhypothese, dass die Erwartungswerte fr alle Effektfunktionen gleich null sind. Im Screenshot (Analyse von Daten zu Beispiel 5) hat die Treatment-Variable drei Ausprgungen, dementsprechend gibt es zwei Effektfunktionen [\(\small g1(K)\) und \(\small g2(K)\)] die laut der Nullhypothese beide im Durchschnitt den Wert Null annehmen. Dies knnte auch wie folgt ausgedrckt werden: \(\small E(g0) = E(g1) = E(g2)\). In der Gleichungs-Schreibweise ist leichter nachvollziehbar, woher sich die Anzahl der Freiheitsgrade (\(\small df=2\)) ergibt (Die Freiheitsgerade errechnen sich durch die Anzahl miteinander verglichener Gruppen minus eins).
Wird der Test der Nullhypothese signifikant (\(\small p < 0.05\)) muss die Hypothese verworfen werden und man kann davon ausgehen, dass mindestens ein durchschnittlicher Treatmenteffekt vorhanden ist, wird der Test nicht signifikant kann die Nullhypothese angenommen werden (unter der Annahme, dass die Power der Analyse ausreichend gro ist).

Bemerke: Dass die Effektfunktionen im Durchschnitt den Wert Null annehmen, bedeutet nicht, dass es keine bedingten Treatmenteffekte gibt (dies wird von Hypothesentest 4 getestet). Es kann auch sein, dass sich der Effekt des Treatments unter einer Bedingung von \(\small K\) gegenlufig zum Effekt des Treatments unter einer anderen Bedingung \(\small K\) verhlt und somit im Durchschnitt kein Effekt gefunden wird. Zur Verdeutlichung dieses Tatsache ein Beispiel: Es kann sein, dass eine Behandlung fr Personen mit Krankheit A wirksam ist, fr Personen mit Krankheit B jedoch schdlich. Ist dies der Fall, kann es passieren, dass die Behandlung fr eine gemischte Gruppe von Personen mit den Krankheiten A oder B keinen durchschnittlichen Effekt aufweist, aber sehr Wohl bedingte Effekte auftreten.

Hypothesentest 2

Die Hypothesentests 2 und 3 testen, ob die Kovariate(n) Einflsse auf die abhngige Variable haben. Hypothesentest 2 testet, ob ein solcher Einfluss in der Referenzgruppe vorliegt, Hypothesentest 3 prft ob ein solcher Einfluss in Form von Interaktionen mit der Treatment-Variablen in den anderen Gruppen vorkommt.

Hypothesentest 2 testet, ob es in der Kontrollgruppe Einflsse der Kovariaten gibt. Gibt es keine Einflsse (dies entsprche der getesteten Nullhypothese) so ist der Erwartungswert der Referenzgruppe unter allen Bedingungen von \(\small K\) identisch (bzw: \(\small g0(K)\) ist konstant). Fr das im Screenshot oben abgebildete Beispiel mit zwei mglichen Kovariaten-Ausprgungen liee sich die Nullhypothese also wie folgt aufschreiben: \(\small E(Y|X=0,K=0) = E(Y|X=0,K=1)\). Da zwei Gruppen miteinander verglichen werden, haben wir (\(\small df= 2-1\)) einen Freiheitsgrad. Alternativ knnte man die Nullhypothese auch wie folgt ausdrcken: \(\small g_{110} = 0\) Fr das Beispiel im Screenshot wird der Hypothesentest signifikant, das heit die Nullhypothese kann verworfen werden, mit einer Wahrscheinlichkeit von 96,01% (\(\small 1-\)p-value) verndert sich der Erwartungswert des Gesundheitsscores (abhngige Variable) auf Populationsebene in der Gruppe die keine Behandlung erhlt (\(\small X=0\)) in Abhngigkeit der Ausprgung der Kovariaten auf Populationsebene.

Hypothesentest 3

Die Nullhypothese des dritten Tests geht davon aus, dass die Kovariate(n) in der Referenzgruppe die gleichen Auswirkungen hat/haben, wie in den anderen Gruppen. Sie geht also davon aus, dass es keine Interaktionen zwischen Kovariate(n) und Treatmentvariable gibt. In Beispiel 5 (siehe Screenshot) lsst sich dies wie folgt mit Erwartungswerten ausdrcken: \(\small E(Y|X=1,K=0) = E(Y|X=1,K=1)\) und \(\small E(Y|X=2,K=0) = E(Y|X=2,K=1)\). Wir haben also zwei Vergleiche zwischen zwei Gruppen, woraus sich die Anzahl der Freiheitsgrade [\(\small df= 2\times (2-1)=2\)] ableiten lsst.
Alternativ lsst sich die Nullhypothese auch wie folgt ausdrcken: \(\small g110 = g210 = 0\). Wird die Nullhypothese signifikant muss sie verworfen werden und es kann davon ausgegangen werden, dass es mindestens eine Interaktion zwischen Kovariate(n) und Treatmentvariable gibt, wird der Test nicht signifikant, kann bei ausreichender Teststrke davon ausgegangen werden, dass keine Interaktionen vorliegen.

Werden Hypothesentest 2 und Hypothesentest 3 beide nicht signifikant, kann bei ausreichender Teststrke davon ausgegangen werden, dass die Kovariate gar keinen Effekt auf die abhngige Variable ausbt: Hypothesentest 2 testet ob es einen Kovariaten-Effekt in der Referenzgruppe gibt, Hypothesentest 3 testet, ob der Effekt in der Referenzgruppe identisch zum Effekt in den anderen Gruppen ist. Werden beide Tests nicht signifikant bedeutet dies also: Es gibt keinen Effekt in der Referenzgruppe (Test 2) und in den anderen Gruppen sieht das genauso aus (Test 3).

Hypothesentest 4

Anders als in Hypothesentest 1 werden hier keine durchschnittlichen Effekte sondern bedingte Effekte gegen Null getestet. Die Nullhypothese ist also erfllt, wenn alle Effektfunktionen unter allen Ausprgungen der Kovariaten gleich null sind. Im Screenshot oben (Analyse von Beispiel 5) gibt es zwei Effektfunktionen und zwei Ausprgungen von \(\small K\). Getestet wird also ob \(\small g1(K) = g2(K) = 0\) sowohl unter der Bedingung \(\small K=0\), als auch unter der Bedingung \(\small K=1\) gilt.
In anderen Worten: es wird getestet, ob der Erwartungswert der abhngigen Variable unter der Bedingung \(\small K=0\) und unter der Bedingung \(\small K=1\) jeweils unter allen Bedinungen von \(\small X\) identisch ist, es wird getestet ob gilt: \(\small E(Y|X=0, K=0) = E(Y|X=1, K=0) = E(Y|X=2, K=0)\) und \(\small E(Y|X=0, K=1) = E(Y|X=1, K=1) = E(Y|X=2, K=1)\). In dieser zweiten Schreibweise lsst sich auch besser erkennen, warum der Test vier Freiheitsgrade hat (\(\small df=4\)): Die Freiheitsgerade errechnen sich wieder durch die Anzahl miteinander Verglichener Gruppen minus 1, somit ergeben sich zwei Freiheitsgrade aus dem Vergleich der drei Erwartungswerte unter der Bedingung \(\small K=0\) und zwei weiere Freiheitsgrade aus dem Vergleich der drei Erwartungswerte unter der Bedingung \(\small K=1\).
Eine dritte Mglichkeit die Nullhypothese fr Hypothesentest 4 zu beschreiben gibt es auf Ebene der Regressionsparameter: \(\small g1(K) = g2(K) = 0\) ist auch dann erfllt, wenn alle Parameter innerhalb der Effektfunktionen gleich Null sind, wenn also gilt: \(\small g_{100} = g_{110} = g_{200} = g_{210} = 0)\).

Wird der Test signifikant (\(\small p < 0.05\)) muss die Hypothese verworfen werden und man kann davon ausgehen, dass mindestens einer der Parameter der Effektfunktionen ungleich null ist bzw. dass mindestens eine Ausprgung der Treatmentvariable \(\small X\) unter mindestens einer der Ausprgungen der Kovariate \(\small K\) einen Effekt auf die abhngige Variable \(\small Y\) ausbt.

Adjusted Means

Wie bei der einfachen linearen Regression werden Mittelwerte fr die Ausprgung der abhngigen \(\small Y\)-Variablen unter verschiedenen Bedingungen von \(\small X\) angezeigt. Die adjustierten Mittelwerte lassen sich im Fall der bedingten linearen Regression genauso interpretieren, wie im Hauptoutput der einfaktoriellen Regression, da ber alle weiteren Kovariaten gemittelt wird. Die durchschnittlichen Effekte lassen sich im Fall der bedingten linearen Regression genauso interpretieren, wie im Hauptoutput der einfaktoriellen Regression erklrt, da ber den zweiten Faktor, die Kovariate, gemittelt wird: Es werden die durchschnittlichen Effekte der Treatmentvariablen geschtzt.
